# openforecaster.github.io

Minimal project page (static HTML/CSS/JS).

## Edit

- Links: `assets/site.config.js`
- Text/layout: `index.html`
- Styling: `assets/styles.css`
- Figures: `assets/media/` (swap files, keep names or update `index.html`)

## Local preview

From `Website/openforecaster.github.io`:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
