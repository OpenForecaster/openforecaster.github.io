# Forecasting Skills & Strategy

## Core Principles
1. **Prioritize early-resolving questions** - They contribute more to TW-score
2. **Submit predictions ASAP** - Don't wait for perfect information
3. **Use broad distributions** for uncertain questions rather than leaving blank (blank = 0 score)
4. **Research pays off** for questions with clear news evidence (like the Syria/SDF question)

## Research Approach
1. Search news for each question topic using mcp__forecast__search_news
2. Focus on recent articles (from_date parameter)
3. Cross-reference multiple sources when possible
4. Key sources: Al Jazeera, Reuters, CBS News, Independent, Straits Times

## Calibration Notes
- For specific person/place names: Use 2-4 candidates with moderate probabilities
- For binary-ish outcomes: Lean toward more likely outcome with 0.6-0.7
- For sports outcomes: Be realistic about underdogs
- For political events: Consider recent trends and power dynamics

## Time Management
- Don't spend more than 2-3 minutes researching any single question
- Use background agents for batch submissions
- Focus research on questions where evidence can shift predictions significantly
